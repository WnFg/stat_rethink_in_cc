# /setup — 课程初始化

目标：**摸底学生背景 → 生成轻量总大纲 → 初始化记忆文件 → 交代后续怎么上课**。10–15 分钟。
读 `CLAUDE.md` 守教学规则。语言：中文为主 + 英文术语。

## 流程

### 1. 摸底（不纠错，只观察，约 5 分钟）
用对话问 4–5 个问题，判断起点：
- 数学背景：微积分/线性代数/概率分布（正态、二项）熟悉程度？
- 编程：Python / NumPy 熟练度？跑过统计/数据代码吗？
- 统计基础：接触过频率派（p 值、置信区间）吗？听过贝叶斯吗？
- 目标与节奏：为什么学？每周能投入多少时间？
- 让学生用一句话回答："如果抛一枚不知道是否均匀的硬币，抛 3 次得到 2 正，你怎么估计它正面的概率？"——借此感知直觉起点（不评判对错）。

### 2. 诊断
归纳：数学/编程/统计 三档起点、3 条优势、3 条易卡点。把画像写进 `progress/state.md` 的"学情画像"。
若摸底中暴露明显概念误区（如"概率就是频率""贝叶斯就是加先验的玄学"），在 `progress/error-log.md` 种 1–3 条 `active` 误区。

### 3. 生成总大纲 `curriculum/syllabus.md`（轻量、一次性）
读全书目录核对页码：`python3 tools/pdftext.py --toc`。
按 **15 章 → 小节 → 计划课次** 列表，每行给：小节号、标题、PDF 页区间、一句话学习目标、动手工具（numpy / scipy / pymc）。每章末给一个**里程碑**（学生能独立做出的事，如"第 2 章：能用网格近似手写出 globe tossing 的 posterior 并解读"）。
分阶段（建议）：
- **阶段一 基础与推断引擎**（ch1–3）：贝叶斯更新、网格近似、从 posterior 采样
- **阶段二 线性回归家族**（ch4–7）：高斯模型、多元回归、过拟合/信息准则、交互
- **阶段三 计算与 GLM**（ch8–11）：MCMC、最大熵/GLM、计数与分类、混合模型
- **阶段四 多层与进阶**（ch12–15）：多层模型、协方差、缺失数据、收尾
不要在 syllabus 里写详细教案——那是 `/chapter` 的事。保持轻量、能一屏扫完每章。

### 4. 初始化 progress 文件
确认这些存在且表头就绪（已有模板就保留）：`state.md`、`error-log.md`、`concept-log.md`、`cheatsheet.md`。
更新 `state.md`：状态=active、当前章=1、下一步=先上第 1 章导论或直接 `/chapter 2`。

### 5. git（可选备份）
若当前不是 git 仓库，建议 `git init` 并首次 `git add . && git commit -m "课程初始化"`。问一下学生要不要。

### 6. 交接
用中文简述：课程概览、四个阶段、第 1 章讲什么、PyMC 到第 8 章前再装。
告诉学生下一步：`/chapter 2` 生成第 2 章教案，然后 `/lesson` 开始上课。第 1 章是纯概念导论，可在这里顺带口头过一遍核心思想（统计模型像 golem、三种工具）。
