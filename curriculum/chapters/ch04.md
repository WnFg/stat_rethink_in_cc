## 4.1 Why Normal Distributions are Normal（PDF p85–89）

- **问题引入（承上启下）**：
  第 2 章用二项式 + 网格近似做推断，第 3 章从后验采样汇总——但两章的"计量"（measurements）都是计数（水面次数）。现实世界里身高、体重、测量误差等**连续变量**怎么建模？这些量的分布惊人地都像"钟形曲线"——为什么？理解来源，才知道什么时候**不该用**高斯。

- **作者核心观点（精华，带页码）**：
  - "Linear regression is the geocentric model of applied statistics… it can usefully describe a very large variety of natural phenomena… if we read its structure too literally, we're likely to make mistakes." (p84)
  - 三种路径都走向高斯：**加法**（大量随机步长之和 → CLT，p85–86）；**乘法**（小效应相乘 ≈ 相加，p87）；**对数乘法**（大效应相乘，在对数尺度上 → 高斯，p88）。
  - 两类辩护：① **本体论**（物理世界里高斯分布广泛存在）；② **认识论**（当我们只愿意说"均值和方差已知"，高斯是最大熵的、最保守的选择）(p88–89)。后者在第 9 章深入。
  - "Using a model is not equivalent to swearing an oath to it." (p89) —— 模型是工具，不是信仰。

- **学习目标**：用加法/乘法/对数乘法三条路径直觉解释高斯的来源，并能把"CLT=路径数最多的地方是零波动"翻译给别人听。

- **核心概念**：
  - **正态分布 Normal / Gaussian distribution**：连续变量上的概率密度函数；均值 μ、标准差 σ 完全刻画（p85–89）。
  - **中心极限定理直觉 CLT intuition**（§4.1.1, p85–86）：大量随机步长相加，各方向波动互相抵消，最终"落在零的方式最多"——所以高斯不是特殊规律而是**极限平凡性**。
  - **本体论 vs 认识论辩护**（§4.1.4, p88–89）：前者"世界里真的到处是高斯"；后者"已知均值/方差时高斯是最大熵分布"。

- **关键直觉**：
  "A large positive fluctuation will cancel a large negative one. The more terms in the sum, the more chances for each fluctuation to be canceled… eventually the most likely sum will be a sum in which every fluctuation is canceled, a sum of zero (relative to the mean)." (p86–87) ——**加和的路径数在零处最多**，钟形就是路径计数的结果——和第 2 章分叉路径花园一模一样！

- **Worked example**（§4.1.1, R code 4.1 的 Python 等价，p85–86）：
  随机游走 16 步（每步 Uniform(-1,1)），1000 人的最终位置分布 → 近正态。

  ```python
  # §4.1.1 正态分布——加法路径 (p85-86)
  import numpy as np
  from scipy import stats

  rng = np.random.default_rng(42)
  N_people, N_steps = 10000, 16

  # 每人独立走 16 步，每步 Uniform(-1, 1)
  steps = rng.uniform(-1, 1, (N_people, N_steps))
  positions = steps.sum(axis=1)

  # 理论：Uniform(-1,1) 方差=1/3，16 步之和的方差=16/3，std≈2.31
  theoretical_std = np.sqrt(N_steps / 3)
  empirical_mean = positions.mean()
  empirical_std  = positions.std()

  assert abs(empirical_mean) < 0.1, f"均值应≈0，实际={empirical_mean:.3f}"
  assert abs(empirical_std - theoretical_std) < 0.3, \
      f"std 应≈{theoretical_std:.2f}，实际={empirical_std:.2f}"

  # Shapiro-Wilk 正态检验（取 1000 个子样本，避免大 n 问题）
  _, pval = stats.shapiro(rng.choice(positions, 1000, replace=False))
  assert pval > 0.01, f"分布显著非正态？p={pval:.3f}"

  print(f"步数={N_steps}: mean={empirical_mean:.3f}, std={empirical_std:.3f}"
        f" (理论={theoretical_std:.3f})")
  print(f"Shapiro-Wilk p={pval:.3f} → 接受正态假设")
  ```

## 4.2 A Language for Describing Models（PDF p90–91）

- **问题引入（承上启下）**：
  上一节弄清楚了"为什么用高斯"。但怎么**把一个具体的贝叶斯模型写清楚**、让别人也看懂？纯文字描述太模糊，R/Python 代码太具体——本节学"中间层"：一套通用的**模型规范符号**，既和 Bayes 定理对得上，又可直接转成代码。

- **作者核心观点（精华，带页码）**：
  - "Scientists increasingly use this same language to describe their statistical methods… learning this language is an investment." (p90)
  - 模型的五步决策（p90）：① 确定结果变量；② 为每个结果变量定义**似然分布**；③ 确定预测变量；④ 将似然的形状与预测变量关联（命名所有参数）；⑤ 给所有参数选先验。
  - "We no longer have to remember seemingly arbitrary lists of bizarre conditions like homoscedasticity, because we can just read these conditions from the model definitions." (p90) —— 模型规范语言让假设变得**显式可读**。
  - 随机关系（~, stochastic）vs 确定关系（=, deterministic）的区分（p91）。

- **学习目标**：能把文字描述的贝叶斯模型翻译成"~ / = + 先验"的数学规范，并反向翻译回来。

- **核心概念**：
  - **随机关系 stochastic（~）**：变量 ~ 分布，表示它的值是不确定的，由分布描述（p91）。
  - **确定关系 deterministic（=）**：变量由其他量完全确定，无随机性（p91，线性模型中出现）。
  - **模型规范语言**（§4.2, p90–91）：
    ```
    yi  ~ Normal(μ, σ)      # 似然
    μ   ~ Normal(0, 10)     # μ 的先验
    σ   ~ Uniform(0, 10)    # σ 的先验
    ```
    与之前 globe tossing 的 `w ~ Binomial(n, p); p ~ Uniform(0,1)` 完全同构（p91）。

- **关键直觉**：
  每行要么是 **likelihood**（第一行），要么是 **prior**（后续行）。从规范到 Bayes 定理是机械的：后验 ∝ likelihood × 所有先验之积（p91，Overthinking 框）。**读懂规范 = 读懂所有假设**（§4.2.1, p91）。

- **Worked example**（§4.2.1 Re-describing globe tossing, p91，R code 4.6 的 Python 等价）：

  ```python
  # §4.2 模型规范 → Bayes 计算（globe tossing 重述，p91）
  import numpy as np
  from scipy.stats import binom, uniform

  w, n = 6, 9
  p_grid = np.linspace(0, 1, 100)

  # 模型规范：
  #   w ~ Binomial(n, p)    [似然]
  #   p ~ Uniform(0, 1)     [先验]
  likelihood = binom.pmf(w, n, p_grid)
  prior      = uniform.pdf(p_grid, 0, 1)          # 均匀先验，pdf=1
  posterior  = likelihood * prior
  posterior /= posterior.sum()

  # MAP ≈ w/n = 6/9 ≈ 0.667
  p_map = p_grid[posterior.argmax()]
  assert abs(p_map - 6/9) < 0.02, f"MAP 应≈6/9≈0.667，得 {p_map:.3f}"
  print(f"globe tossing MAP = {p_map:.3f}  (理论 MLE = {6/9:.3f})")
  ```

## 4.3 A Gaussian Model of Height（PDF p92–103）

- **问题引入（承上启下）**：
  学会了"用高斯"和"写模型规范"，现在动真格：用真实人类身高数据做一个**双参数贝叶斯模型**（μ, σ 同时推断）。这是第一次面对"参数空间不是一维网格"的问题——如何做二维网格近似、为什么要转向 MAP 二次近似？

- **作者核心观点（精华，带页码）**：
  - "There are an infinite number of possible Gaussian distributions… We want our Bayesian machine to consider every possible distribution… and rank them by posterior plausibility." (p92)
  - 模型：hi ~ Normal(μ, σ); μ ~ Normal(178, 20); σ ~ Uniform(0, 50)。作者 178cm 的身高是 μ 先验中心的来源！(p95)
  - "A distribution of distributions. Yes. A distribution of distributions." (p92) —— 后验是 (μ, σ) 的联合分布，对应无数条可能的高斯分布。
  - i.i.d. 假设（§4.3.2, p94）：不是本体论声明（"测量真的独立"），而是**认识论假设**（"我对相关性一无所知"）——这是 Jaynes 的"mind projection fallacy"警告（p94）。
  - 先验预测（§4.3.2, p95–96）：在看数据之前，先从先验模拟可能的身高分布，检验先验是否合理。这个习惯贯穿全书。
  - MAP 二次近似（§4.3.5, p99）：爬山找 MAP，在峰附近用高斯近似——和第 2 章的 quadratic approximation 思路完全相同，只是现在是二维。

- **学习目标**：
  1. 会用 NumPy 做二维网格近似（双参数后验）；
  2. 会用 `scipy.optimize.minimize` 做 MAP 估计 + 用 Hessian 得协方差矩阵；
  3. 会从多维后验中采样并汇总（HPDI/均值）。

- **核心概念**：
  - **Howell1 数据**（§4.3.1, p92–93）：!Kung San 人口普查，544 人，4 列（height cm / weight kg / age / male）；本节只用成人（age≥18, 352 人）。
  - **联合后验 joint posterior**（§4.3.2, p92）：Pr(μ, σ | heights) ∝ ∏i Normal(hi|μ,σ) × Normal(μ|178,20) × Uniform(σ|0,50)。
  - **MAP 二次近似**（§4.3.5, p99）：`scipy.optimize.minimize(neg_log_posterior)` → MAP；逆 Hessian ≈ 后验协方差矩阵。
  - **方差-协方差矩阵 variance-covariance matrix**（§4.3.6, p102–103）：多维高斯的完整描述；`cov2cor` 提取相关矩阵。

- **关键直觉**：
  - **网格近似 → MAP 二次近似的必要性**：双参数 200×200 = 40000 个格点，已可计算；10 个参数就是 200^10 = 无穷——网格不可行，必须用近似。MAP 找峰，Hessian 描述峰的"曲率=精度"（p99）。
  - **先验预测**（p95–96）：从 Normal(178,20) 和 Uniform(0,50) 各采样，合成高斯，看"高斯族的高斯"长什么样——峰在 178 附近但很宽，告诉你先验很弱。

- **Worked example A — 数据加载 + 先验预测**（§4.3.1–4.3.2, p92–96，R code 4.7–4.13 的 Python 等价）：


- **Worked example B — 网格近似双参数后验**（§4.3.3, p96–97，R code 4.14–4.18 的 Python 等价）：

  ```python
  # §4.3.3 二维网格近似 (p96-97)
  import numpy as np
  import pandas as pd
  from scipy import stats
  import os

  DATA_PATH = 'materials/data/Howell1.csv'
  d = pd.read_csv(DATA_PATH, sep=';')
  d2 = d[d['age'] >= 18].reset_index(drop=True)
  heights = d2['height'].values  # (352,)

  mu_list    = np.linspace(140, 160, 100)   # 100 × 100 格点
  sigma_list = np.linspace(4, 9, 100)
  MU, SIGMA  = np.meshgrid(mu_list, sigma_list)  # 形状 (100, 100)

  # 向量化对数似然：heights[:, None, None] 广播到 (352, 100, 100)
  log_lik = np.sum(
      stats.norm.logpdf(heights[:, None, None], MU[None], SIGMA[None]),
      axis=0
  )  # 形状 (100, 100)

  log_prior_mu    = stats.norm.logpdf(MU, 178, 20)          # Normal(178,20)
  log_prior_sigma = np.where((SIGMA > 0) & (SIGMA <= 50), 0.0, -np.inf)  # Uniform(0,50)

  log_post = log_lik + log_prior_mu + log_prior_sigma
  log_post -= log_post.max()        # 数值稳定
  post = np.exp(log_post)
  post /= post.sum()                 # 归一化

  # 网格 MAP
  idx = np.unravel_index(post.argmax(), post.shape)
  mu_map_grid    = MU[idx]
  sigma_map_grid = SIGMA[idx]

  assert 154.0 < mu_map_grid < 156.0, f"μ MAP 应≈154.6，实际={mu_map_grid:.2f}"
  assert 7.0   < sigma_map_grid < 9.0, f"σ MAP 应≈7.7， 实际={sigma_map_grid:.2f}"
  print(f"网格 MAP: μ={mu_map_grid:.2f} cm, σ={sigma_map_grid:.2f} cm")

  # 从后验中采样（边缘分布）
  flat_post = post.ravel()
  flat_post /= flat_post.sum()
  rng = np.random.default_rng(42)
  idx_samp = rng.choice(len(flat_post), size=10000, p=flat_post)

  sample_mu    = MU.ravel()[idx_samp]
  sample_sigma = SIGMA.ravel()[idx_samp]
  print(f"后验样本: μ mean={sample_mu.mean():.2f}±{sample_mu.std():.2f},"
  ```

- **Worked example C — MAP 二次近似 + 协方差矩阵**（§4.3.5–4.3.6, p99–103，R code 4.24–4.32 的 Python 等价）：

  ```python
  # §4.3.5-4.3.6 MAP 二次近似 (p99-103)
  import numpy as np
  import pandas as pd
  from scipy import stats
  from scipy.optimize import minimize
  import os

  DATA_PATH = 'materials/data/Howell1.csv'
  d = pd.read_csv(DATA_PATH, sep=';')
  d2 = d[d['age'] >= 18].reset_index(drop=True)
  heights = d2['height'].values

  # 负对数后验（使用 log_sigma 参数化确保 σ>0）
  def neg_log_post(params):
      mu, log_sigma = params
      sigma = np.exp(log_sigma)
      if sigma <= 0 or sigma > 50:
          return np.inf
      ll    = np.sum(stats.norm.logpdf(heights, mu, sigma))
      lp_mu = stats.norm.logpdf(mu, 178, 20)
      # Uniform(0,50) 先验：log density 在范围内为常数，不影响优化
      return -(ll + lp_mu)

  result = minimize(neg_log_post, x0=[170.0, np.log(8.0)], method='BFGS')
  mu_map    = result.x[0]
  sigma_map = np.exp(result.x[1])

  # 数值 Hessian → 后验协方差（BFGS hess_inv 近似在 352 个数据点时精度不足）
  eps = 1e-4
  x0  = result.x
  H   = np.zeros((2, 2))
  for ii in range(2):
      for jj in range(2):
          xpp = x0.copy(); xpp[ii] += eps; xpp[jj] += eps
          xpm = x0.copy(); xpm[ii] += eps; xpm[jj] -= eps
          xmp = x0.copy(); xmp[ii] -= eps; xmp[jj] += eps
          xmm = x0.copy(); xmm[ii] -= eps; xmm[jj] -= eps
          H[ii, jj] = (neg_log_post(xpp) - neg_log_post(xpm)
                       - neg_log_post(xmp) + neg_log_post(xmm)) / (4 * eps**2)
  cov_log  = np.linalg.inv(H)   # 在 (μ, log_σ) 空间
  se_mu    = np.sqrt(cov_log[0, 0])
  # delta method: SE(σ) = SE(log_σ) × σ
  se_sigma = np.sqrt(cov_log[1, 1]) * sigma_map

  # 对应书中结果 (p100): μ≈154.61±0.41, σ≈7.73±0.29
  assert 154.0 < mu_map < 156.0,    f"μ MAP 应≈154.6，实际={mu_map:.2f}"
  assert 7.0   < sigma_map < 8.5,   f"σ MAP 应≈7.7， 实际={sigma_map:.2f}"
  assert 0.25  < se_mu < 0.65,      f"SE(μ) 应≈0.41，实际={se_mu:.3f}"
  assert 0.15  < se_sigma < 0.50,   f"SE(σ) 应≈0.29，实际={se_sigma:.3f}"
  print(f"MAP: μ={mu_map:.2f}±{se_mu:.2f},  σ={sigma_map:.2f}±{se_sigma:.2f}")
  print(f"（书中: μ=154.61±0.41, σ=7.73±0.29, p100）")
  ```

## 4.4 Adding a Predictor（PDF p105–122）

- **问题引入（承上启下）**：
  §4.3 只用了**身高**一个变量（只有截距）。现实中我们关心"知道某人体重，能不能预测身高"——这就是**回归**的核心。本节引入**线性模型策略**：把均值 μ 变成预测变量的线性函数，让数据"教"参数（截距 α 和斜率 β）。

- **作者核心观点（精华，带页码）**：
  - "The strategy is to make the parameter for the mean… into a linear function of the predictor variable." (p105) —— **线性模型策略** = 让 μ 依赖于 x。
  - "Nothing special or natural about linear models… you can choose a different relationship." (p106) —— 线性只是习惯，不是物理真理。
  - 参数的意义（p107）：α = 当 xi=0 时的期望 yi（截距，常常无物理意义）；β = xi 变化一单位，yi 的期望变化（斜率）。
  - **先验选择哲学**（p108, Rethinking 框）："There is no more a uniquely correct prior than there is a uniquely correct likelihood." —— 先验编码了看数据之前的知识状态，可以随之讨论、探索。
  - **中心化 centering**（p112–113）：使 α 可解释（= 预测变量取均值时的期望结果）；同时消除 α、β 的高度负相关，数值更稳定。
  - **两类不确定性**（PDF p122, Rethinking 框 "Two kinds of uncertainty"）：①参数不确定性（后验均值的 HPDI）；②观测不确定性（预测区间，含 σ）。必须区分！

- **学习目标**：
  1. 能写出线性模型的数学规范并翻译成 Python；
  2. 用 MAP + 后验采样计算任意权重处的均值 μ_i 的分布；
  3. 区分"均值的 HPDI"（参数不确定）和"预测区间 PI"（观测不确定），并可视化。

- **核心概念**：
  - **线性模型**（§4.4.1, p105–107）：
    ```
    hi   ~ Normal(μi, σ)      # 似然（μ 现在随个体变化）
    μi    = α + β·xi           # 线性模型（确定关系）
    α    ~ Normal(178, 100)   # 截距先验
    β    ~ Normal(0, 10)      # 斜率先验（0 中心 = 保守）
    σ    ~ Uniform(0, 50)     # σ 先验
    ```
  - **截距 α / 斜率 β intercept/slope**（p107）：α 是 xi=0 时的期望高度；β 是每增加 1 kg 体重，期望增加多少 cm 高度。
  - **中心化 centering**（§4.4.3.1, p112–113）：`x_c = x - mean(x)`，使 α = 数据均值时的期望 y，且 Corr(α, β) ≈ 0。
  - **均值不确定性 vs 预测不确定性**（§4.4.3.4–4.4.3.5, p115–122）：均值 HPDI = 只看 (α,β) 的后验；预测 PI = 还加上 σ 的随机性（`Normal(μ_i, σ)`）。

- **关键直觉**：
  - **后验是"无数条直线"的分布**（p113–116）：每一对 (α, β) 定义一条直线；后验给每条直线一个可信度权重。画 20 条采样直线，就能看到"哪里确定、哪里不确定"（图 4.5, p116）。
  - **直线两端不确定性最大**（p116–117）：预测变量离均值越远，均值 HPDI 越宽——这是因为所有直线都"从数据密集区穿过"，到边缘则发散。

- **Worked example A — 线性模型 MAP 估计 + 中心化**（§4.4.1–4.4.3.1, p105–113，R code 4.38–4.44 的 Python 等价）：

  ```python
  # §4.4 线性回归 MAP 估计 + 中心化 (p105-113)
  import numpy as np
  import pandas as pd
  from scipy import stats
  from scipy.optimize import minimize
  import os

  DATA_PATH = 'materials/data/Howell1.csv'
  d = pd.read_csv(DATA_PATH, sep=';')
  d2 = d[d['age'] >= 18].reset_index(drop=True)
  heights = d2['height'].values
  weights = d2['weight'].values

  # 负对数后验（线性模型）
  def neg_log_post_lm(params):
      a, b, log_sigma = params
      sigma = np.exp(log_sigma)
      mu_i  = a + b * weights
      ll    = np.sum(stats.norm.logpdf(heights, mu_i, sigma))
      lp_a  = stats.norm.logpdf(a, 178, 100)
      lp_b  = stats.norm.logpdf(b, 0, 10)
      return -(ll + lp_a + lp_b)

  res = minimize(neg_log_post_lm, [114, 0.9, np.log(5)], method='BFGS')
  a_map, b_map, sigma_map = res.x[0], res.x[1], np.exp(res.x[2])

  # 书中 (p111, R code 4.40): a≈113.90, b≈0.90, sigma≈5.07
  assert 111.0 < a_map < 117.0, f"α MAP 应≈113.9，实际={a_map:.2f}"
  assert 0.84  < b_map < 0.96,  f"β MAP 应≈0.90，实际={b_map:.2f}"
  assert 4.7   < sigma_map < 5.4, f"σ MAP 应≈5.07，实际={sigma_map:.2f}"
  print(f"线性模型 MAP: α={a_map:.2f}, β={b_map:.2f}, σ={sigma_map:.2f}")

  # 中心化：weight_c = weight - mean(weight)
  weight_mean = weights.mean()
  weights_c   = weights - weight_mean

  def neg_log_post_lm_c(params):
      a, b, log_sigma = params
      sigma = np.exp(log_sigma)
      mu_i  = a + b * weights_c
      ll    = np.sum(stats.norm.logpdf(heights, mu_i, sigma))
      lp_a  = stats.norm.logpdf(a, 178, 100)
      lp_b  = stats.norm.logpdf(b, 0, 10)
      return -(ll + lp_a + lp_b)

  res_c = minimize(neg_log_post_lm_c, [154, 0.9, np.log(5)], method='BFGS')
  a_c, b_c, sigma_c = res_c.x[0], res_c.x[1], np.exp(res_c.x[2])

  # 中心化后 α≈mean(height)≈154.6，β 不变
  assert abs(a_c - heights.mean()) < 1.0, \
      f"中心化后 α 应≈均值身高={heights.mean():.1f}，实际={a_c:.2f}"
  assert abs(b_c - b_map) < 0.02, "中心化不改变 β 估计"
  print(f"中心化后 α={a_c:.2f} ≈ 均值身高={heights.mean():.1f} cm")
  print(f"→ α 现在有直接意义：体重=均值时的期望身高（书 p112–113）")
  ```

- **Worked example B — 后验预测可视化（均值 HPDI + 预测 PI）**（§4.4.3.3–4.4.3.5, p116–122，R code 4.46–4.62 的 Python 等价）：
  > 📌 教材 §4.4.3.2（p113–114, 图 4.4）有一步"先画 MAP 直线"——本教案有意跳过，直接到后验采样，请理解这是压缩步骤，概念上 MAP 直线 = 后验采样均值线（R code 4.45）。

  ```python
  # §4.4.3.3-4.4.3.5 后验采样 → 均值 HPDI + 预测 PI (p113-122)
  import numpy as np
  import pandas as pd
  from scipy import stats
  from scipy.optimize import minimize
  import os

  DATA_PATH = 'materials/data/Howell1.csv'
  d = pd.read_csv(DATA_PATH, sep=';')
  d2 = d[d['age'] >= 18].reset_index(drop=True)
  heights = d2['height'].values
  weights = d2['weight'].values

  def neg_log_post_lm(params):
      a, b, log_sigma = params
      sigma = np.exp(log_sigma)
      mu_i  = a + b * weights
      ll    = np.sum(stats.norm.logpdf(heights, mu_i, sigma))
      lp_a  = stats.norm.logpdf(a, 178, 100)
      lp_b  = stats.norm.logpdf(b, 0, 10)
      return -(ll + lp_a + lp_b)

  res = minimize(neg_log_post_lm, [114, 0.9, np.log(5)], method='BFGS')

  # 从多维高斯后验中采样（协方差来自逆 Hessian）
  rng = np.random.default_rng(42)
  post_samples = rng.multivariate_normal(res.x, res.hess_inv, size=10000)
  a_samp     = post_samples[:, 0]
  b_samp     = post_samples[:, 1]
  sigma_samp = np.exp(post_samples[:, 2])

  # 在 weight=50 处的 μ 后验分布
  mu_at_50 = a_samp + b_samp * 50
  hpdi_lo  = np.percentile(mu_at_50, 5.5)
  hpdi_hi  = np.percentile(mu_at_50, 94.5)
  # 书中 (p117): 89% 区间≈[158.6, 159.7]
  assert 157.0 < mu_at_50.mean() < 161.0, \
      f"weight=50 时 E[μ]≈159, 实际={mu_at_50.mean():.2f}"
  print(f"weight=50 的 μ: mean={mu_at_50.mean():.2f}, 89% CI=[{hpdi_lo:.1f},{hpdi_hi:.1f}]")

  # 在一系列 weight 值处的均值 HPDI 和预测 PI
  w_seq  = np.arange(25, 71)
  mu_mat = a_samp[:, None] + b_samp[:, None] * w_seq[None, :]  # (10000, 46)
  mu_mean  = mu_mat.mean(axis=0)
  mu_hpdi_lo = np.percentile(mu_mat, 5.5, axis=0)
  mu_hpdi_hi = np.percentile(mu_mat, 94.5, axis=0)

  # 预测区间：在 μ_i 上加正态随机噪声 σ_i
  h_sim = rng.normal(mu_mat, sigma_samp[:, None])  # (10000, 46)
  pi_lo = np.percentile(h_sim, 5.5, axis=0)
  pi_hi = np.percentile(h_sim, 94.5, axis=0)

  # 预测区间应包含均值区间，且明显更宽
  assert np.all(pi_lo <= mu_hpdi_lo + 1),  "预测区间下界应≤均值 HPDI 下界"
  assert np.all(pi_hi >= mu_hpdi_hi - 1),  "预测区间上界应≥均值 HPDI 上界"
  width_mu = (mu_hpdi_hi - mu_hpdi_lo).mean()
  width_pi = (pi_hi - pi_lo).mean()
  assert width_pi > width_mu * 3, f"预测区间应远宽于均值区间（{width_pi:.1f} vs {width_mu:.1f}）"
  print(f"均值 HPDI 平均宽={width_mu:.1f} cm | 预测 PI 平均宽={width_pi:.1f} cm")
  print("（预测区间包含了 σ 的观测不确定性，书 p120–122）")
  ```

    ```python
    # solution: ch4 ex 4.4-2 (呼应 p113-115)
    import numpy as np
    import pandas as pd
    from scipy import stats
    from scipy.optimize import minimize
    import os

                 + stats.norm.logpdf(a, 178, 100)
    ```

## 4.5 Polynomial Regression（PDF p123–127）

- **问题引入（承上启下）**：
  §4.4 的线性模型（直线）在成人数据上表现不错。但如果包含儿童（全部 544 人），height vs weight 明显是**弯曲的**（体重从 5 kg 到 65 kg，身高非线性增长）。怎么用"线性模型家族"拟合曲线？多项式回归是一个答案——但作者明确警告：**这是工具，不是解释**。

- **作者核心观点（精华，带页码）**：
  - "In general it's a bad thing to do. Polynomials are very hard to interpret." (p123) —— 多项式是 geocentric 描述；有条件用，别沉迷。
  - "Better would be to have a more mechanistic model of the data… built from a principled beginning." (p123) —— 真正的好模型应有理论支撑（例如 log 体重回归，4H3 题）。
  - 标准化 standardizing（p124–125）：先中心化再除以标准差。两个原因：① 解释更自然（一单位 = 一个 SD）；② 避免数值溢出（大数的平方/三次方 → 数值炸了）。
  - "We should feel embarrassed to use them, just so we don't become satisfied with the phenomenological explanations they provide." (p125) —— 用多项式可以，但要保持批判意识。

- **学习目标**：
  1. 能对预测变量做标准化（centering + scaling）；
  2. 会在 Python 中拟合抛物线（二次多项式）回归并可视化；
  3. 明白多项式回归是工具而非解释，知道其局限。

- **核心概念**：
  - **多项式回归 polynomial regression**（§4.5, p123）：
    ```
    μi = α + β1·xi + β2·xi²        # 抛物线（二次）
    ```
    线性模型仍然是参数的线性函数（"linear in parameters"），只是 x 本身被加了高次项（p124–125）。
  - **标准化 standardizing**（§4.5, p124）：`x_s = (x - mean(x)) / sd(x)`，使预测变量均值=0、标准差=1；比单纯中心化更彻底。
  - **β₁（线性项）vs β₂（曲率项）**（p125–127）：β₂<0 → 开口向下的抛物线（成人+儿童：轻时快速长高，重时趋缓）。

- **关键直觉**：
  多项式 = 添加更多"自由度"来拟合弯曲——但自由度越多，曲线越"随数据起伏"，越危险（第 6 章的 overfitting 预警）。第 4 章展示怎么用，第 6 章告诉你为什么要小心。

- **Worked example**（§4.5, p123–127，R code 4.65–4.68 的 Python 等价）：用全部 544 人数据拟合抛物线回归。

  ```python
  # §4.5 多项式回归（二次，p123-127）
  import numpy as np
  import pandas as pd
  from scipy import stats
  from scipy.optimize import minimize
  import os

  DATA_PATH = 'materials/data/Howell1.csv'
  d = pd.read_csv(DATA_PATH, sep=';')
  heights = d['height'].values    # 全部 544 人（含儿童）
  weights = d['weight'].values

  # 标准化：weight_s = (weight - mean) / std
  w_mean, w_std = weights.mean(), weights.std()
  ws  = (weights - w_mean) / w_std          # weight standardized
  ws2 = ws ** 2                              # 二次项

  def neg_log_post_poly(params):
      a, b1, b2, log_sigma = params
      sigma = np.exp(log_sigma)
      mu_i  = a + b1 * ws + b2 * ws2
      ll    = np.sum(stats.norm.logpdf(heights, mu_i, sigma))
      lp_a  = stats.norm.logpdf(a, 178, 100)
      lp_b1 = stats.norm.logpdf(b1, 0, 10)
      lp_b2 = stats.norm.logpdf(b2, 0, 10)
      return -(ll + lp_a + lp_b1 + lp_b2)

  res = minimize(neg_log_post_poly, [120, 20, -8, np.log(6)], method='BFGS')
  a_m, b1_m, b2_m, sigma_m = res.x[0], res.x[1], res.x[2], np.exp(res.x[3])

  # 书中 (p125, R code 4.67): a≈146.66, b1≈21.40, b2≈-8.42, sigma≈5.75
  assert 140 < a_m < 155,    f"α 应≈146.66，实际={a_m:.2f}"
  assert 18  < b1_m < 25,    f"β₁ 应≈21.40，实际={b1_m:.2f}"
  assert -12 < b2_m < -5,    f"β₂ 应≈-8.42（负=开口向下），实际={b2_m:.2f}"
  assert 4.5 < sigma_m < 7,  f"σ 应≈5.75，实际={sigma_m:.2f}"
  print(f"多项式 MAP: α={a_m:.2f}, β₁={b1_m:.2f}, β₂={b2_m:.2f}, σ={sigma_m:.2f}")
  print(f"（书中: α=146.66, β₁=21.40, β₂=-8.42, σ=5.75, p125）")
  print(f"β₂={b2_m:.2f}<0 → 抛物线开口向下：轻时成长快、重时趋缓")

  # 在一系列标准化 weight 处做后验均值预测
  rng = np.random.default_rng(42)
  post_p = rng.multivariate_normal(res.x, res.hess_inv, 10000)
  ws_seq = np.linspace(-2.2, 2.0, 50)

  mu_pred = (post_p[:, 0:1]
             + post_p[:, 1:2] * ws_seq[None, :]
             + post_p[:, 2:3] * ws_seq[None, :] ** 2)  # (10000, 50)

  mu_mean  = mu_pred.mean(axis=0)
  mu_lo    = np.percentile(mu_pred, 5.5, axis=0)
  mu_hi    = np.percentile(mu_pred, 94.5, axis=0)

  # 均值曲线应在合理范围内（身高 50–200 cm）
  assert mu_mean.min() > 50 and mu_mean.max() < 200, \
      f"预测均值超出合理范围：[{mu_mean.min():.1f}, {mu_mean.max():.1f}]"
  print(f"预测身高范围（均值）: [{mu_mean.min():.1f}, {mu_mean.max():.1f}] cm")
  ```

  - **练习 4.5-1**：将抛物线回归换成**三次多项式**（cubic, μi = α + β₁·xi + β₂·xi² + β₃·xi³），重新拟合全部 544 人数据（书中 R code 4.70, p127）。报告 MAP 估计值。检查：β₃ 的符号与大小，三次曲线比抛物线"更弯"吗（书 p127）？（呼应 §4.5, p126–127）
    <details><summary>解法与预期答案</summary>

    ```python
    # solution: ch4 ex 4.5-1 (呼应 p126-127, R code 4.70)
    import numpy as np
    import pandas as pd
    from scipy import stats
    from scipy.optimize import minimize
    import os

    DATA_PATH = 'materials/data/Howell1.csv'
    d = pd.read_csv(DATA_PATH, sep=';')
    heights = d['height'].values
    weights = d['weight'].values
    w_mean, w_std = weights.mean(), weights.std()
    ws  = (weights - w_mean) / w_std
    ws2 = ws ** 2
    ws3 = ws ** 3

    def neg_log_post_cubic(params):
        a, b1, b2, b3, log_sigma = params
        sigma = np.exp(log_sigma)
        mu_i  = a + b1*ws + b2*ws2 + b3*ws3
        ll    = np.sum(stats.norm.logpdf(heights, mu_i, sigma))
        return -(ll + stats.norm.logpdf(a, 178, 100)
                    + stats.norm.logpdf(b1, 0, 10)
                    + stats.norm.logpdf(b2, 0, 10)
                    + stats.norm.logpdf(b3, 0, 10))

    res = minimize(neg_log_post_cubic, [147, 21, -8, 0, np.log(5)], method='BFGS')
    a_m, b1_m, b2_m, b3_m, sigma_m = (res.x[0], res.x[1], res.x[2],
                                        res.x[3], np.exp(res.x[4]))

    print(f"三次多项式 MAP: α={a_m:.2f}, β₁={b1_m:.2f}, β₂={b2_m:.2f},"
          f" β₃={b3_m:.2f}, σ={sigma_m:.2f}")
    # σ 应比二次更小（拟合更好）
    poly2_sigma = 5.75   # 前面练习得到的二次多项式 σ
    assert sigma_m < poly2_sigma + 0.5, "三次多项式 σ 应不大于二次（拟合更好或持平）"
    # β₃ 的意义
    print(f"β₃={'正' if b3_m > 0 else '负'}（方向解释依赖三次项的物理含义）")
    ```

  - ❌ "多项式拟合更好 → 模型更正确" → ✅ 训练集拟合好不等于正确（第 6 章 overfitting）；多项式是 geocentric，描述≠解释（p123–124）。
