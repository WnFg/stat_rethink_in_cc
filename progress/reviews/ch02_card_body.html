② 数学思想与作者核心观点 ★

McElreath 在第 2 章真正想让你接受的，是一套统计世界观——比任何公式都重要。每条都可追溯到原文。

小世界 vs 大世界（small world / large world） — 模型只是"小世界"里自洽的逻辑；它在小世界最优，不保证在真实的"大世界"也好。大世界表现只能事后检验，无法由逻辑推出。 §引言, p32
贝叶斯推断没有玄学，就是"数路径" — "Bayesian inference is really just counting and comparing of possibilities." 乘法只是压缩了的计数；Bayes 定理之所以看着复杂，只因它把"数路径"这件事的关系藏起来了。 §2.1, p33 / §2.3, p50
没有免费午餐（no free lunch） — 贝叶斯估计对任何样本量都有效，代价是依赖先验；频率派则要选估计量。"Both golems pay for lunch with their assumptions."——两边都用假设买单，没有谁不付钱。 §2.2 Rethinking, p44
模型不是"真理"，关键问"够不够用" — 模型的假设永远不完全对，所以"检验模型是否为真"没意义；目标是检查它对某个用途是否够用。"Failure to conclude that a model is false must be a failure of our imagination, not a success of the model." §2.2 evaluate, p43–45
模型很自信 ≠ 模型很好 — 数据越多后验越尖，但再自信也只是"在这个模型设定下"；错误的模型一样可以很自信。 §2.2, p43
先验是"工程假设"，是模型的一部分 — 先验不神秘也不主观可耻，它和似然一样是可被检验、修改、质询的建模选择。"interrogated like other assumptions: by altering it and checking how sensitive inference is." §2.3.3, p48–49
概率是"认知"，不是客观实在 — de Finetti："PROBABILITY DOES NOT EXIST." 概率描述的是一个信息有限的观察者对世界的不确定，而非世界本身的客观属性。说"水的概率是 0.7"，说的是我们的知识状态。 §2.6 习题 2E4, p59
贝叶斯分析不等于"用了 Bayes 定理"；怎么拟合也是模型的一部分 — 任何概率推断最终都会用到 Bayes 定理；贝叶斯的特别之处是用它量化对不可观测量（参数/模型）的不确定。而且"How you fit the model is part of the model"——网格/二次/MCMC 的选择会影响结论。 §2.3 Rethinking, p50 / §2.4 Rethinking, p52

关键数学思想（动机，不只公式）

  • 乘法 = 压缩的计数：先验×似然之所以是"相乘"，是因为路径数 = 先验路径数 × 新路径数（§2.1.2, p38）。
  • 归一化 = 把相对计数变成概率：分母"平均似然"只为让后验积分到 1，不含额外信息（§2.3.4, p50）。
  • 对数后验 ≈ 抛物线：二次近似的全部数学动机——任何高斯的对数是抛物线，于是峰附近用二阶泰勒就够（§2.4.2, p54）。
  • 后验是分布，不是点：MLE/MAP 只是峰；贝叶斯坚持给出整条不确定性，这是它对小样本"诚实"的根源（§2.4.2, p57）。

③ 核心概念与公式

四个核心词汇（§2.1, p40）

parameter 参数 — 要推断的未知量 p；代码里是 grid 上每个候选值。
likelihood 似然 — 给定 p 产生数据的相对可能性，binom.pmf(W,N,p)。
prior 先验 — 看数据前对 p 的初始权重，np.ones_like(grid)。
posterior 后验 — 看完数据后的权重 Pr(p|W)，一整条分布。

Bayes 定理（§2.3, p49–50）

后验 = (似然 × 先验) / 平均似然
Pr(p|W) = Pr(W|p)·Pr(p) / Pr(W)　　Pr(W)=∫Pr(W|p)Pr(p)dp =(like×prior).sum()

二次近似（§2.4.2, p55）

对数后验 —(二阶泰勒，近似)→ 抛物线 —(exp，精确)→ 高斯　|　μ=MAP，σ²=1/|f″|，SD=1/√曲率

⑤ 三种引擎对比（§2.4, p52）

引擎怎么做特点
① Grid 网格近似把 p 切网格，逐点算 先验×似然 归一化直观、教学用；随参数增多急剧变慢
② Quadratic 二次近似用高斯逼近后验峰（MAP+SD）便宜、后半本书主力；小样本/偏态失真
③ MCMC不直接算，从后验抽样复杂/多层模型主力（第 8 章）

MLE 与 MAP

MLE = argmax 似然 = 观测频率 W/N；MAP = argmax(似然×先验)。均匀先验时 MAP=MLE；先验有倾向时 MAP 被拉偏；数据越多 MAP→MLE。两者都只是"一个点"，贝叶斯真正的答案是整条后验。

🎯 待强化：可信区间 ≠ 置信区间（练到脱口而出）

贝叶斯·可信区间
"给定数据，p 落在 [a,b] 的概率是 95%。" 概率属于参数：P(a≤p≤b|数据)=0.95。
频率派·置信区间
参数是固定值；95% 指程序——重复实验 95% 的此类区间盖住真值。不能对单个区间说"95% 含真值"。

# 网格近似：算后验（均匀先验）
grid = np.linspace(0, 1, 1000)
like = binom.pmf(W, N, grid)        # 似然（grid=一排候选 p）
post = like / like.sum()            # .sum() 就是平均似然 Pr(W)=分母
map_p = grid[np.argmax(post)]       # MAP（均匀先验下=MLE=W/N）

# 95% 可信区间（百分位）
cum = np.cumsum(post)
lo, hi = grid[np.searchsorted(cum,0.025)], grid[np.searchsorted(cum,0.975)]

# 二次近似：找峰 + 量曲率
neg = lambda p: -binom.logpmf(W, N, np.clip(p,1e-9,1-1e-9))
m   = minimize(neg, 0.5, method="L-BFGS-B", bounds=[(1e-6,1-1e-6)]).x[0]
sd  = 1/np.sqrt((neg(m+eps)-2*neg(m)+neg(m-eps))/eps**2)
