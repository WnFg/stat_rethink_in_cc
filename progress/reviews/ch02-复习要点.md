## 1. 核心概念精要

### 1.1 数路径花园（§2.1, p33–40）
- 每个未知取值是一条岔路；观测**剪掉**不符的路径；剩下路径数的相对大小 = 可信度。
- **路径数相乘**（各层），再 **÷ 总和标准化**成概率。
- 四个词（p40）：**parameter** 参数 / **likelihood** 似然(相对路径数) / **prior** 先验 / **posterior** 后验。

### 1.2 建模与贝叶斯更新（§2.2, p41–44）
- globe tossing：估水比例 `p`，观测 `W L W W W L W L W`=6W3L。离散弹珠 → 连续 `p`，逻辑不变。
- **三步**：data story → update → evaluate。
- **贝叶斯更新**：从先验起逐数据更新；见 W 峰右移、见 L 左移；数据越多越窄越高；**与顺序无关**(p44)。

### 1.3 四零件 + Bayes 定理（§2.3, p45–50）
```
              似然          先验
   Pr(p|W) = Pr(W|p)   ×   Pr(p)
   后验      ─────────────────────
                  Pr(W)  ← 平均似然 = 归一化分母
```
- **平均似然 Pr(W)** = `Σ(似然×先验)`（离散）= `∫like·prior dp`（连续），作用只是让后验和为 1。
- 即你代码里 `post / post.sum()` 的那个 `.sum()`。

### 1.4 三种引擎（§2.4, p50–57）
| 引擎 | 做法 | 备注 |
|------|------|------|
| **grid 网格近似** | 切网格逐点算 `先验×似然` 归一化 | 教学利器；参数一多就爆炸(p54) |
| **quadratic 二次近似** | 峰附近用高斯逼近：**MAP + SD** | 后面半本书主力(p55) |
| **MCMC** | 从后验**抽样** | 复杂模型用，第 8 章 |

## 2. 🔍 你的兴趣点 · 深入解答（你主动追问的，集中收好）

### 2.1 MLE 是什么？和 MAP 什么关系？
- **MLE（最大似然估计）** = 让观测数据最可能的参数 = **似然曲线的峰**。二项分布里 **MLE = 观测频率 W/N**。
- **MAP（后验众数）** = **(似然×先验) 的峰**。
- 关系：
  ```
  MLE = argmax  Pr(data|p)            只看似然
  MAP = argmax  Pr(data|p)·Pr(p)      似然×先验
  ```
  - **均匀先验 → MAP = MLE**（乘常数不动峰）。
  - 先验有倾向 → 先验把峰拉偏（例：偏 0.5 的先验把 6/9=0.667 拉到 0.53）。
  - 数据越多 → MAP → MLE（先验被冲淡）。
- ⚠️ 二者都只是**一个点（峰）**，丢掉了整条分布；贝叶斯真正的产物是**整条后验**。

### 2.2 二次近似的数学基础（你最关心的一题）
**核心链**：
```
后验 ≈ 高斯  ⟺  对数后验 ≈ 抛物线  （因为"任何高斯的对数 = 抛物线"是精确事实）
```
**做法**：对【对数后验 f(p)】在峰 p̂ 处做二阶泰勒展开——
```
f(p) ≈ f(p̂) + f'(p̂)(p−p̂) + ½f''(p̂)(p−p̂)²
峰处 f'(p̂)=0 → 只剩 f(p̂) + ½f''(p̂)(p−p̂)²   ← 抛物线
exp 还原 → 高斯：μ = p̂，σ² = 1/|f''(p̂)|
```
**`σ²=1/|f''|` 怎么来的**：把"近似后验" `exp(−½|f''|(p−p̂)²)` 和"高斯" `exp(−½(1/σ²)(p−μ)²)` 并排，**比较 (p−中心)² 前的系数**：`|f''| = 1/σ²`，取倒数即得。
- **直觉**：`|f''|` = 曲率 = 精度 = `1/σ²`。**峰越尖 → 方差越小 → 越确定**。
- 多维：`f''` → Hessian 矩阵，`σ²` → 负 Hessian 的逆（协方差矩阵）。
- 何时准：大样本后验趋高斯(Bernstein–von Mises)；**小样本/偏态失真**（n=9 时右尾溢到 p≈1 不可能区）。
- **关键**：被"近似"的是【对数后验→抛物线】这一步；【高斯↔抛物线】是**精确**的对数关系，不是近似。

### 2.3 "很多故事对应同一个模型"（§2.2, p42）
- 模型是**数学骨架**（如 Binomial），故事是**因果机制**。globe / 抛硬币 / A-B 测试 / 质检 都给同一个 `Binomial(N,p)`。
- 推论：**模型拟合好 ≠ 你的因果故事对**（数据骨架相同，故事可能错）。→ 区分因果要靠额外工具（第 5、6 章）。

### 2.4 几个 API（你问过的，速记）
- `binom.pmf(k, n, p)`：n 次试验恰好 k 次成功的概率。**p 传数组(grid)= 一次算所有候选 p 的似然**（向量化）。
- `binom.logpmf(...)`：上面的对数版，防小数连乘下溢、且 quap 理论在对数尺度。
- `np.clip(a, lo, hi)`：把数值夹进 [lo, hi]（护栏）。quap 里 `np.clip(p,1e-9,1-1e-9)` 防 `log(0)=-∞`。
- `minimize(f, x0, method="L-BFGS-B", bounds=...)`：从 x0 出发找 f 的最小值。求后验**最大** = 求**负**对数后验**最小**（`min(-f)=max(f)`）。`res.x[0]` 取出最优 p = MAP。

## 3. ⚠️ 你的薄弱点 · 重点盯防（来自练习中卡过/答偏的地方）

| # | 易错 | 正确 | 记忆锚点 |
|---|------|------|---------|
| 1 | 分母/后验里乘 `grid`(p 值) | 乘 **`prior`(先验权重)** | 分母 = Σ(似然 × **先验**)，不是 ×参数值 |
| 2 | `neg(m+e) - neg(m) / e` | `(neg(m+e) - neg(m)) / e` | Python 里 `/` 优先级高于 `−`，**先减要加括号** |
| 3 | "用抛物线近似后验，再取对数得高斯" | 近似的是**对数后验**→抛物线；**exp**(非 log)得高斯 | 近似只在"对数后验→抛物线"那一步 |
| 4 | 把可信区间说成"95% 置信区间" | 见下方 ⭐ | 概率属于**参数**不属于**程序** |

### ⭐ 可信区间 vs 置信区间（务必能脱口而出）
- **贝叶斯可信区间 credible interval**："给定数据和模型，**P(p ∈ [0.43, 0.82] | 数据) = 0.95**"——概率是关于**参数**的。
- **频率派置信区间 confidence interval**：参数固定，95% 指**程序**——"重复实验很多次，95% 的区间会盖住真值"；**不能**说"这一个区间里有 95% 概率含真值"。

## 4. 🧰 可复用 Python 模板（本章四件套）

```python
import numpy as np
from scipy.stats import binom

# ① 网格近似后验（grid approximation）
def grid_posterior(W, N, n_grid=1000, prior=None):
    p = np.linspace(0, 1, n_grid)
    prior = np.ones_like(p) if prior is None else np.asarray(prior, float)
    post = binom.pmf(W, N, p) * prior
    return p, post / post.sum()

# ② MAP + 95% 可信区间（百分位区间）
def summarize(p, post):
    cum = np.cumsum(post)
    lo, hi = p[np.searchsorted(cum, 0.025)], p[np.searchsorted(cum, 0.975)]
    return p[np.argmax(post)], (lo, hi)          # (MAP, 95%CI)

# ③ 二次近似（quadratic approximation）
from scipy.optimize import minimize
def quap(W, N):
    neg = lambda q: -binom.logpmf(W, N, np.clip(q, 1e-9, 1-1e-9))   # 负对数后验
    m = minimize(neg, 0.5, method="L-BFGS-B", bounds=[(1e-6, 1-1e-6)]).x[0]
    e = 1e-4
    second = (neg(m+e) - 2*neg(m) + neg(m-e)) / e**2               # 曲率 |f''|
    return m, 1/np.sqrt(second)                                    # (MAP, SD)

# ④ MLE（点估计）= 似然峰 = W/N
# mle = grid[np.argmax(binom.pmf(W, N, grid))]
```
